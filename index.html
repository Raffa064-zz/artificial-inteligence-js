Artificial Inteligence

About

With artificial-inteligence.js module, you can get an simple way to use MLPs with javascript.

This module are implemented only for reinforcement learn, and I do not intend to implement supervised learning algorithms.

Tests

You can test current version by seen the execution logs here

The current features

The current version include:

  • Tanh activation function;
  • Crossing over method (Simple weight and bias split);
  • Mutation method (Random mutation).

NOTE: This module are written for reinforcement learn... Also I will not add any implementation of backpropagation.

Future features

I'm thinking to add more features for future versions like:

  • More activation functions;
  • Train enviroment structure;
  • Import/Export to JSON.

How to install and use

Download

For use this module you can copy the code in the github repository, or intall module file by terminal:

Linux and Windows

You can use wget for download the module into current directory, if you dont have wget intalled install it with your prefered package manager like apt:

$ wget https://raffa064.github.io/artificial-inteligence-js/js/artificial-inteligence.js

Mac

You can use curl for download the module into current directory:

$ curl https://raffa064.github.io/artificial-inteligence-js/js/artificial-inteligence.js

Script tag on html

You can also add the module from script tag in your html:

<script src="https://raffa064.github.io/artificial-inteligence-js/js/artificial-inteligence.js"></script>

How to instanciate an NeuralNetwork

For instanciate your network manually you will need to import theNeuralNetwork class from artificial-inteligence.js module:

//If you have added the module by script tag, it will not be necessary
import NeuralNetwork from '<YOUR COPY OF MODULE PATH>'

After, you will need to create an instance of the class, given for her these data:

  • The number of inputs of the network;
  • The number of hidden layers;
  • The number of neurons per hidden layer;
  • The number of neurons in the output layer.

The below code creates a network with 2 inpus, 2 hidden layers, 2 neurons per hidden layer and 1 output layer:

...
const network = new NeuralNetwork(1, 2, 2, 1)
...
